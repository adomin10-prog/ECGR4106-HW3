## Assignment

This project implements sequence-to-sequence machine translation models for ECGR 4106 Homework 3. The notebook includes:

- English-to-French baseline GRU model
- English-to-French GRU model with attention
- French-to-English reverse-direction models
- Training and validation metrics
- BLEU score evaluation
- Attention visualization
- Custom sentence testing
